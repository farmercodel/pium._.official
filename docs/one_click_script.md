# 원클릭 실행 스크립트

## :computer: Mac/Linux

`./setup.sh`
초기 설정 스크립트 (한번만 실행하면 됨)

`./start-dev.sh`
개발 컨테이너 실행 스크립트

`./logs.sh`
로그 확인 스크립트
스크립트 입력 후 1~4 원하는 컨테이너 숫자 확인 후 입력

`./stop.sh`
개발 컨테이너 중지 스크립트
